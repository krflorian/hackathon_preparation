# Logistic Hackthon Vorbereitung 

## get started 
- create venv 
```shell 
pip install venv 
python -m venv .env
```

- activate venv  

powershell: 
```shell 
.env/Scripts/Activate.ps1
```

cmd: 
```shell 
env/Scripts/activate.bat
```

- install requirements 
```shell 
pip install -r requirements.txt
```

- solve hill_climbing.ipynb
